# SEC-021: CORS wildcard configuration

| 字段 | 值 |
|---|---|
| 规则编号 | SEC-021 |
| 分类 | 安全 (Security) |
| 严重等级 | 🟠 High |

## 为什么这是问题

配置 CORS（跨域资源共享）为 `*` 通配符（或 `Access-Control-Allow-Origin: *`）意味着任何网站都可以向你的 API 发起跨域请求。这完全违背了浏览器同源策略的初衷，使你的 API 暴露给所有第三方网站。如果 API 依赖于 Cookie、Session 或 Authorization Header 进行身份验证（且设置了 `Access-Control-Allow-Credentials: true`），通配符配置将允许任何恶意网站代表已认证用户发起请求（CSRF 攻击）。

即使 API 是公开的，`*` 配置也过于宽松。攻击者可以在自己的恶意网站上放置利用你 API 的脚本，收集用户数据或利用 API 功能。对于需要用户认证的 API，CORS 应严格限制为受信任的域名。

## 修复方法

1. **指定精确的允许来源**：列出前端应用的确切域名，不要使用通配符。
2. **对于多环境部署**：根据环境动态设置允许的来源。
3. **限制允许的方法和头**：只开放 API 实际需要的方法和请求头。
4. **不要同时使用 `*` 和 `credentials: true`**：这是浏览器明确禁止的组合，但错误的配置仍然可能被忽略。
5. **在开发环境使用工具代理**：而不是在生产 API 上配置宽松的 CORS。

## 示例

```javascript
// ❌ 错误的做法：CORS 配置过于宽松
const cors = require("cors");

app.use(cors());  // 默认允许所有来源

// 或者
app.use(cors({
    origin: "*",                 // ← 允许任何域名
    methods: ["GET", "POST", "PUT", "DELETE", "PATCH"],
    credentials: true            // ← 同时允许凭证，违反规范
}));

// ❌ 另一个常见错误
app.use((req, res, next) => {
    res.header("Access-Control-Allow-Origin", "*");
    res.header("Access-Control-Allow-Credentials", "true");  // 危险组合！
    next();
});
```

```javascript
// ✅ 正确的做法：限制为受信任的域名
const cors = require("cors");

const allowedOrigins = [
    "https://myapp.com",
    "https://www.myapp.com",
    "https://admin.myapp.com",
];

app.use(cors({
    origin: function (origin, callback) {
        // 允许无来源的请求（如 Postman、移动端）
        if (!origin) return callback(null, true);
        if (allowedOrigins.includes(origin)) {
            callback(null, true);
        } else {
            callback(new Error("Not allowed by CORS"));
        }
    },
    methods: ["GET", "POST"],  // 只开放需要的方法
    allowedHeaders: ["Content-Type", "Authorization"],
    credentials: true,         // 仅在限定 origin 后使用
    maxAge: 86400              // 预检请求缓存 24 小时
}));
```

```python
# ✅ Python/Flask 正确配置
from flask_cors import CORS

app = Flask(__name__)
CORS(app, origins=[
    "https://myapp.com",
    "https://www.myapp.com"
], methods=["GET", "POST"])
```
